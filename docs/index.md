## 🚀 Quick Start

```bash
# Install Dart SDK
# https://dart.dev/get-dart

# Check version
dart --version

# Create a new project
dart create my_app
cd my_app

# Run it
dart run
```

## 💡 Recommended Learning Paths

Choose the path that best matches your experience level to get the most out of this tutorial:

*   **🌱 Absolute Beginners:** Start at the very beginning with [Introduction to Dart](./introduction.md) and progress sequentially through each chapter to build a strong foundation.
*   **🚀 Experienced Developers:** Skim the basics and dive straight into the unique features of Dart, such as [Null Safety](./null-safety.md), [Records](./records.md), and [Pattern Matching](./pattern-matching.md).
*   **💙 Flutter Developers:** Ensure you cover [Functions](./functions.md), [Async & Futures](./futures-and-async.md), and our specialized [Flutter-Specific Tips](./flutter-tips.md) to supercharge your app development.
*   **⚡ In a Hurry?** Head directly to the [Cheat Sheet](./cheat-sheet.md) for a comprehensive syntax reference on a single page.
